## Dashboard

`dashboard.html` is a self-contained interactive HTML file showing per-participant SGIO recommendations from the NHANES validation. Open it directly in any modern browser — no server required.

**Add this file:** copy `smgil_dashboard_scaled.html` from your local results folder into this directory and rename it `dashboard.html`.
